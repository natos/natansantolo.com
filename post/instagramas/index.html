Instagramas

A component to show the latest Instagram pictures.

I built this simple component for Marcela’s Website, the idea is to show the latest pictures you’ve published in your Instagram.

I tried to make it flexible by adding a few configurations, you can choose the type of render, amount of images, also some flags to hide tags and favorites. See all the possible configurations and examples.

I also made a test page where you can find more example implementations and some notes about styling. Also if you like to hack things go and grab the code in github.

I’ll keep improving this little project, its quite fun and there’s quite a lot of opportunities. As soon as I make some time, the first thing I want to fix is the CSS implementation and documentation.
